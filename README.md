# 專案名稱 - Mentor

這是一個假想的工程師職涯互助平台，用戶可以在上面註冊帳號，並成為導師，並接收其他用戶傳來的留言。
用戶也可以根據不同的程式語言、專長、導師所在地區等條件，搜尋符合條件的導師。
另外也可以與其他用戶建立即時聊天室，並能夠同時與多人聊天。

```bash

運行開發環境
